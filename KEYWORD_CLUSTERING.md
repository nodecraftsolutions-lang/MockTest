This document contains comprehensive keyword research, clustering, and targeting strategy for the PrepZon online mock test platform.

## Keyword Research Methodology

### Tools Used
- Google Keyword Planner
- Google Trends
- Competitor Analysis
- Search Console Data (future)
- User Search Intent Analysis

### Keyword Selection Criteria
1. **Relevance:** Directly related to PrepZon's offerings
2. **Search Volume:** Minimum 100 monthly searches
3. **Competition:** Balance of high and low competition
4. **User Intent:** Aligned with user goals (informational, navigational, transactional)
5. **Conversion Potential:** Keywords likely to convert

## Keyword Optimization Guidelines

### Title Tag Formula
```
[Primary Keyword] - [Benefit/Feature] | PrepZon
```
**Examples:**
- Online Mock Tests - Practice with Real Exam Patterns | PrepZon
- Company-Specific Tests - Ace Your Placement Exam | PrepZon

### Meta Description Formula
```
[Action Verb] [Primary Keyword] on PrepZon. [Benefit 1], [Benefit 2], and [Benefit 3]. [Call to Action]
```
**Example:**
"Take online mock tests on PrepZon. Practice with company-specific patterns, get instant results, and compete with thousands of students. Start practicing free today!"

### H1 Tag Formula
```
[Primary Keyword] + [Unique Value Proposition]
```
**Examples:**
- Online Mock Tests That Mirror Real Exam Patterns
- Company-Specific Practice Tests for Guaranteed Success

### Content Keyword Density
- **Primary Keyword:** 1-2% density
- **Secondary Keywords:** 0.5-1% density
- **LSI Keywords:** Natural throughout
- **Long-Tail Variations:** In subheadings and body
